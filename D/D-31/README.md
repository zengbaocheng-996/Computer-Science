### MIME

![](2.png)

SMTP的缺点：

1. SMTP不能传送可执行文件或者其他二进制对象。
2. SMTP仅限于传送7位ASCII码，不能传送其他非英语国家的文字
3. SMTP服务器会拒绝超过一定长度的邮件

使电子邮件系统可以支持声音、图像、视频、多种国家语言等等。
