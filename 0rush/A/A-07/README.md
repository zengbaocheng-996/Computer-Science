# 栈与队列的应用

### 队列的应用

1. 解决逐行或逐层的问题，如层序遍历二叉树。
2. 解决主机与外部设备之间速度不匹配的问题，如缓冲区。
3. 解决由多用户引起的资源竞争问题，如进程的就绪队列。

### 栈在递归中的应用

函数调用时，需要用一个栈存储：

1. 调用返回地址
2. 实参
3. 局部变量

注：理论上所有的递归算法都可以手动建栈，用非递归的方式实现。

越靠后被调用的函数信息越靠近栈顶

某个函数return后该函数在栈中的信息就会消失
