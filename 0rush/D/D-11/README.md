### 以太网交换机的两种方式

1. 直通式交换机

   查完目的地址（6B）就立刻转发。

   延迟小，可靠性低，无法支持具有不同速率的端口的交换。

2. 存储转发式交换机

   将帧放入高速缓存，并检查是否正确，正确则转发，错误则丢弃。

   延迟大，可靠性高，可以支持具有不同速率的端口的交换。
